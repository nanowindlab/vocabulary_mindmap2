# Legacy Archive Guide

> Role: 세션 handoff 및 초기 실험 산출물의 legacy history zone

## Scope

이 폴더는 현재 운영 SSOT가 아니다.

포함 대상:

- 오래된 `NEXT_THREAD_HANDOFF_V*.md`
- 초기 재시작 메모
- 구세대 앱/문서 폴더 스냅샷

## Rule

- 이 폴더의 handoff 문서는 모두 history-only 참고 자료로 본다.
- 현재 작업 지시, 현재 상태 판단, 다음 액션 결정에 직접적인 SSOT로 사용하지 않는다.
- 현재 운영 기준은 루트 `README.md`, `PROJECT_DOCUMENT_MAP.md`, `.gemini-orchestration/`, `08_expansion/` canonical 문서다.

## Migration Policy

문서형 구버전 spec/audit는 가능하면 `08_expansion/archive/`로 통합한다.
이 폴더는 당분간 아래 용도로만 유지한다.

- 옛 세션 흐름 복기
- reset 전후 맥락 확인
- `_v2` 시절 산출물 참조
