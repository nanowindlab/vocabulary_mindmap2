# 엄격한 데이터 분류 프로토콜 (Strict Data Classification Protocol, SDCP) V2

> **긴급 공지**: 데이터 에이전트의 1차 전수 분류 결과, **26%의 심각한 오분류**가 적발되었습니다. 이는 기획자가 제공한 IA 정의를 자의적으로 해석하여 발생한 사고입니다. 본 V2 프로토콜은 **[네거티브 가이드(Negative Guide)]**를 최우선으로 적용하며, 모든 모호한 판단은 '보류 큐'로 강제합니다.

---

## 1. [핵심 보완] 네거티브 가이드: 절대 금지 조항 (Strict Exclusions)
데이터 에이전트가 가장 많이 실수한 3대 오분류 유형을 원천 차단합니다.

1.  **[동물/식물] ➔ [사람과 관계] 배치 절대 금지**
    - **오류 사례**: '강아지', '고양이', '가축', '소', '닭'
    - **수정 지침**: 인간이 아닌 모든 생명체는 [사람과 관계]가 아닌 **[날씨와 자연]**으로 강제 배치한다. 반려동물이라 하더라도 '관계' 중심이 아닌 '생물'로서 우선 분류한다.
2.  **[직업/역할] ➔ [사람과 관계] 배치 절대 금지**
    - **오류 사례**: '의사', '교사', '사장', '군인', '종업원'
    - **수정 지침**: 직업 명칭은 그 사람이 활동하는 **'작업 환경(Scene)'** 루트로 우선 배치한다.
        - 의사/간호사 ➔ [보건과 의료]
        - 선생님/학생 ➔ [학교와 공부]
        - 사장/대리 ➔ [직장과 업무]
    - **예외**: 가족(부모/자녀) 및 보편적 사회 호칭(친구/지인)만 [사람과 관계]에 허용한다.
3.  **[장소/건물] ➔ [상황 지시/기타] 배치 절대 금지**
    - **오류 사례**: '병원', '학교', '시장', '우체국'
    - **수정 지침**: 구체적인 건물이나 공공 기관은 반드시 해당 **'상황 루트'**에 꽂혀야 한다. 14번 루트 [상황 지시/기타]는 색깔, 모양, 지시어만 담는 대피소이며, 구체적인 장소가 들어와서는 안 된다.

---

## 2. [강화된 루틴] 3대 축별 문장형 정밀 정의 및 자가 검수 질문

### 2.1. [상황과 장소] (Situations & Places)
- **정의**: "이 단어를 들었을 때 특정한 **'물리적 공간'**이나 **'행동 장면'**이 머릿속에 즉시 그려지는가?"
- **자가 검수**: "이 단어가 없어도 해당 상황(Scene)을 설명하는 데 문제가 없는가? 그렇다면 제외하라."

### 2.2. [마음과 표현] (Heart & Expression)
- **정의**: "이 단어가 사물이나 타인에 대한 **'나의 주관적 느낌이나 평가'** 혹은 **'내면의 감정 상태'**를 나타내는가?"
- **자가 검수**: "단순히 눈에 보이는 사실(객관적 묘사)인가, 아니면 내 생각이 들어간 평가인가? 평가가 아닐 경우 제외하라."

### 2.3. [구조와 기초] (Structure & Basics)
- **정의**: "이 단어가 없으면 **'문장의 뼈대'**가 무너지거나, 논리적 연결이 불가능해지는가?"
- **자가 검수**: "이 단어가 구체적인 그림(명사/동사)을 그리는가? 그렇다면 제외하고 다른 축으로 보내라."

---

## 3. [프로세스 고도화] 보류 큐(Pending Queue) 상신 원칙
**"자의적 판단은 범죄다."** 다음의 경우 무조건 보류 큐로 보낸다.

1.  **다의어 (Polysemy)**: 하나의 단어가 두 개의 탭에 모두 어울리는 경우 (예: '차' - 마시는 차 vs 타는 차)
2.  **분류 불능**: IA V4의 어떤 카테고리(중분류)에도 이름이 딱 들어맞지 않는 경우
3.  **애매한 관계**: '가축'처럼 동물인지 사람의 소유물(일상)인지 헷갈리는 경우

---

## 4. [검수 알고리즘] 데이터 에이전트 자가 검증 리스트
작업 완료 보고 전, 다음 3가지를 전수 체크하십시오.

- [ ] **가축/동물**이 [사람과 관계]에 단 하나라도 있는가? (있다면 즉시 이동)
- [ ] **직업인(의사 등)**이 [사람과 관계]에 있는가? (있다면 즉시 이동)
- [ ] **구체적인 건물/장소**가 [기타] 루트에 있는가? (있다면 즉시 이동)

---
**기획자의 경고**: "26%의 오분류는 IA 시스템의 실패가 아니라, 가이드라인을 무시한 집행의 실패입니다. V2 프로토콜 위반 시 데이터 공정은 즉시 중단될 수 있습니다."
