# 연관 단어(Cross-link) 비약적 확장을 위한 전략 보고서 (V1)

> **핵심 목표**: 현재 2% 미만인 어휘 간 연관 관계(Cross-link) 밀도를 비약적으로 높여, 학습자가 끊김 없이 의미망을 탐색할 수 있는 '초연결 마인드맵' 환경을 구축합니다.

---

## 1. [Strategy 1] 카테고리 기반 시스템 매핑 (Structural Linking)

개별 단어의 매칭 이전에, IA V4의 **루트(Root) 및 카테고리(Category) 간의 상호 유망 관계**를 정의하여 링크 후보군을 대량으로 생성합니다.

### 1.1. 3대 축 간의 핵심 연결 고리 (Tri-Pillar Bridge)
- **[상황] ↔ [마음]**: 특정 상황에서 주로 발생하는 감정이나 상태 연결.
  - 예: `병원/의료` ➔ `신체 통증/증상`, `학교/공부` ➔ `지적 태도/성적 감정`.
- **[상황] ↔ [구조]**: 특정 장소에서 필수적으로 사용되는 단위나 지시어 연결.
  - 예: `쇼핑/결제` ➔ `수량/단위 명사`, `교통/길 찾기` ➔ `방향/위치 지시`.
- **[마음] ↔ [구조]**: 감정의 정도를 표현하는 강조 부사 연결.
  - 예: `내면/감정` ➔ `논리/부연(매우, 무척, 전혀)`.

### 1.2. 실행 방법
- 데이터 에이전트가 특정 카테고리(A)의 단어를 처리할 때, 기획자가 정의한 유망 카테고리(B) 내의 **상위 밴드(Band 1~2) 단어들**을 연관 단어 후보로 자동 추천함.

---

## 2. [Strategy 2] 의미론적 핵심어 자동 매칭 (Semantic Extraction)

어휘의 사전적 정의(Definition) 데이터에서 **명사 키워드**를 추출하여 다른 단어와의 연결 고리를 찾습니다.

### 2.1. 정의 내 키워드 매칭 (Def-Keyword Match)
- **로직**: 단어 A의 정의 문장에 단어 B가 포함되어 있다면, `A ➔ B` 방향의 링크를 자동 생성함.
  - 예: "사과: 사과나무의 **열매**" ➔ `사과` 와 `열매` 자동 연결.
- **다국어 매칭**: 영어 정의(def_en)에서 공통된 핵심어(Core Concept)를 공유하는 단어끼리 연결.
  - 예: 'Shop'과 'Store'를 핵심어로 가진 `가게`, `상점`, `마트` 등을 상호 연결.

---

## 3. [Strategy 3] 학습자 사고 흐름 기반 연결 (Casual Scenario)

사용자가 하나의 단어를 학습할 때 자연스럽게 떠올리는 **'연상 질문'**을 데이터 구조로 전환합니다.

### 3.1. 3대 연상 질문 모델링 (3-Question Model)
1.  **"어디서 써요?" (Where)**: [마음/구조] 단어에서 관련 [상황] 장소로 연결.
    - 예: `맛있다` ➔ `식당/카페`.
2.  **"어떤 느낌이에요?" (How)**: [상황/구조] 단어에서 관련 [마음] 표현으로 연결.
    - 예: `시험` ➔ `어렵다/쉽다`, `떨리다`.
3.  **"얼마나/언제 해요?" (When/How much)**: 모든 단어에서 관련 [구조] 요소로 연결.
    - 예: `커피` ➔ `한 잔(수량/단위)`, `회의` ➔ `지금/아까(시간)`.

---

## 4. [Strategy 4] 외부 언어 자원 및 벌크 데이터 주입

원천 소스(사전) 외의 검증된 언어학적 데이터를 활용합니다.

### 4.1. 관계형 DB 통합
- **유의어/반의어 (Synonym/Antonym)**: 기존 사전의 유/반의어 필드를 전수 스캔하여 `Cross-link` 필드로 전환.
- **상위어/하위어 (Hypernym/Hyponym)**: '과일'과 '사과, 배, 포도' 간의 계층적 링크를 마인드맵 브랜치와 별개로 추가 연결하여 탭 전환을 유도.

---

## 5. 실행 블루프린트 (Data Agent 가이드)

1.  **단계적 주입**: 
    - 1단계: 기존 유/반의어 데이터를 통한 확정적 링크 주입 (밀도 약 5% 확보).
    - 2단계: 카테고리 매핑 규칙에 따른 대량 후보군 생성 및 LLM 정제 (밀도 약 15% 확보).
    - 3단계: 정의 키워드 매칭을 통한 세부 링크 완성 (밀도 최종 20% 이상 목표).
2.  **품질 관리**: 연관 단어는 해당 단어의 **상세 패널 '정의 & 연관' 탭**에 우선 표시하며, 마인드맵 캔버스에는 `Cross-link` 아이콘으로 시각화함.
